# 🏦 Advanced Bank Management System (C++)

A robust, console-based banking application developed in **C++** that demonstrates professional-level software design, secure data handling, and complex logic implementation.

---

## 🚀 Key Features

* **User Management System:** Secure login with multi-user support.
* **Granular Permissions (Bitwise Logic):** A sophisticated permission system using bitwise operations to control access to specific features (Add, Delete, Update, etc.).
* **Complete Client CRUD:** Create, Read, Update, and Delete client records with real-time data persistence.
* **Persistent Data Storage:** Efficient file handling system that ensures all data is saved and retrieved from text files accurately.
* **Input Validation:** Comprehensive handling of user inputs to prevent system crashes and ensure data integrity.

---

## 🛠 Technical Highlights

* **Language:** C++
* **Data Structures:** Advanced use of Vectors and Structs for data organization.
* **Algorithms:** Implementation of searching and sorting algorithms for client management.
* **System Architecture:** Modular code structure separated into logical components for better maintainability.
